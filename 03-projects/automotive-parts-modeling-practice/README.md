# Automotive Parts Modeling Practice

This project includes various mechanical parts used in automotive systems, modeled in SolidWorks. The main purpose of this work is not to present a single production-ready component, but to improve SolidWorks skills, understand automotive part geometry, and gain practical experience in mechanical design.

## Project Purpose

This project was prepared to improve SolidWorks usage and develop a better understanding of automotive mechanical components. During the process, different part types were modeled to practice sketching, feature creation, dimensioning, part editing, and file organization.

## Scope of the Study

This folder contains SolidWorks files of various mechanical parts that can be used in automotive systems. The parts are presented as a general modeling practice rather than separate individual projects.

## Software Used

- SolidWorks

## Project Content

- 3D modeling studies of automotive components
- SolidWorks source files
- Part modeling practice files
- Design studies created for learning and improvement purposes

## Skills Gained

- SolidWorks part modeling practice
- Understanding the basic geometry of automotive components
- Dimensioning and feature creation skills
- Developing technical modeling experience for team projects
- Gaining familiarity with the mechanical design process

## Note

This study was created for educational, portfolio, and team development purposes. The files were prepared to improve SolidWorks skills and to practice modeling automotive-related mechanical components.
