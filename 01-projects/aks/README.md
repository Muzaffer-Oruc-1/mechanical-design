# Aks Tasarımı

Bu projede SolidWorks kullanılarak aks parçası modellenmiştir.

## Proje Amacı
Bu çalışmanın amacı, mekanik bir aks parçasının 3B modellemesini oluşturmak ve tasarım dosyalarını portfolyo olarak düzenli şekilde sunmaktır.

## Kullanılan Program
- SolidWorks

## Proje İçeriği
- 3B modelleme
- Mekanik parça tasarımı
- SolidWorks kaynak dosyaları

## Dosyalar
- Aks.zip: Projeye ait SolidWorks dosyalarını içerir.
