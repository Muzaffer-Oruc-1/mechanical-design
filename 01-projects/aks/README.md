# Wheel Hub, Knuckle and Bearing Assembly

This project focuses on the 3D modeling of wheel hub, steering knuckle, and bearing components used in a vehicle suspension and wheel connection system. The parts were modeled using SolidWorks as part of a mechanical design and automotive component modeling study.

## Project Purpose

The purpose of this project is to create 3D models of key mechanical components used in the wheel connection system and to understand the relationship between the hub, knuckle, and bearing assembly. The project is presented as part of a mechanical design portfolio.

## Software Used

- SolidWorks

## Project Content

- Wheel hub modeling
- Steering knuckle modeling
- Bearing placement and positioning
- Mechanical connection surface design
- SolidWorks source files

## Files

- Right_Knuckle_Hub_Bearing.zip: Contains the SolidWorks project files related to the wheel hub, knuckle, and bearing assembly.

## Skills Gained

- Automotive component modeling practice
- Understanding wheel connection system geometry
- Positioning bearing and hub components
- Creating mechanical connection surfaces
- Organizing SolidWorks part and assembly files
